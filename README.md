# Resolution-week_1-Hardware
### A RC timing circuit featurng 6 Leds in a grid.
This circuit simply turns the Leds one by one, when the power is off they turn off one by one

#### How does it work?
The capacitors act as a powerbank and a delay at the sametime, the diodes prevent the capacitor to charge the previous Led, and the resistor at the start changes the charging speed of the capacitors while the resistors in sereis with the Leds prevent them from burning up.
